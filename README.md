# ⏰ Digital Clock using JavaScript

A simple and responsive **Digital Clock** built using **HTML, CSS, and JavaScript** that displays the current time in real-time. The clock automatically updates every second using JavaScript timing functions.

---

## 🚀 Features
- Displays **current hours, minutes, and seconds**
- Updates **every second automatically**
- Clean and responsive **user interface**
- Built using **pure JavaScript (no libraries)**

---

## 🛠️ Technologies Used
- **HTML** – Structure of the clock  
- **CSS** – Styling and layout  
- **JavaScript** – Logic for updating time dynamically

---

## 📂 Project Structure

```
digital-clock/
│
├── index.html
├── style.css
└── script.js
```

---

## ⚙️ How It Works
The project uses the JavaScript **Date()** object to get the current system time.  
The `setInterval()` function updates the clock **every second**, ensuring the time displayed is always accurate.
